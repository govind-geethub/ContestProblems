# Algorithms & CP Practice in C++

A personal repository of C++ solutions covering Codeforces ladders and its Division Contests, CodeChef Starters Contests, and LeetCode weekly & biweekly challenges — focused on algorithmic improvement, competitive programming practice, and clean, readable code.

---

## 📂 Repository Structure

* `/Codeforces/` — Solutions to ladder problems and live contest rounds and post-contest upsolving.
* `/CodeChef/` — Solutions from Starters contests.
* `/LeetCode/` — Solutions to daily problems and weekly/biweekly contest challenges.
